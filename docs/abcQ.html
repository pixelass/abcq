decode

decode(str) → {Number}

Method to decode a combination of characters into a number

This method does not affect any other method. This method can be called multiple times before calling generate

Parameters:
Name Type Description
str String

Character combination to decode. Must contain only valid characters, Given a list of `"ab" the following will be returned

                 - "a"  -> 0
                 - "b"  -> 1
                 - "aa" -> 2
                 - "ab" -> 3
                 - ...
Returns:

Returns the index of the input string

Type
Number
Example
const shortid = new abcQ({
  chars: 'ab'
})
console.log(shortid.decode('a'))
// -> o
console.log(shortid.decode('abb'))
// -> 9

A number greater than -1. Given a list of `"ab" the following will be returned

             - 3 -> "ab"
             - ...

String

const shortid = new abcQ({
  chars: 'ab'
})
console.log(shortid.encode(0))

console.log(shortid.encode(9))

generate

generate() → {String}

Method to generate the next string.

This method is not affected by calling other methods. It will always return the next combination of characters

Returns:

Returns the next character combination

Type
String
Example
const shortid = new abcQ({
  chars: 'ab'
})
let counter = 0
do {
  console.log(shortid.generate())
} while (++counter < 10)
// -> a b aa ab ba bb aaa aab aba abb
